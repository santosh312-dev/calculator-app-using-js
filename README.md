# 🧮 Simple Web Calculator

A simple calculator built using **HTML, CSS, and JavaScript**.  
It uses JavaScript's `eval()` function to evaluate mathematical expressions.

> ⚠️ Warning: `eval()` is used for simplicity but is not recommended for production due to security risks.

---

## 🚀 Features

- Basic arithmetic operations (+, −, ×, ÷)
- Clear (C) and delete (⌫) buttons
- Instant calculation of expressions
- Responsive design for mobile and desktop
- Simple and beginner-friendly UI

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript (Vanilla JS)

## 💡 How It Works

The calculator takes input as a string and evaluates it using:

```javascript
eval(expression)
```

Example:

* Input: `12+3*2`
* Output: `18`

---

## ⚠️ Security Note

`eval()` can execute any JavaScript code, so it should be avoided in real-world secure applications.

Safer alternatives:

* math.js library
* Custom expression parser
